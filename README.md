# Green Hydrogen Production Optimization using Reinforcement Learning

## Overview
This project applies Reinforcement Learning (PPO) to optimize the operation
of a green hydrogen production plant with storage using real electricity prices.

## Scenarios
- Grid electricity market
- Power Purchase Agreement (PPA)

## Features
- Custom Gymnasium environment
- PPO agent (Stable-Baselines3)
- Finland 2023 spot price data
- Hydrogen storage modeling

## How to Run
Install dependencies:
pip install -r requirements.txt
Run training:
python main.py
## Goal
Learn an adaptive hydrogen production strategy minimizing electricity cost.
